# java-lotto-precourse

## ✅ 기능 구현 목록

- [ ] 사용자에게 입력 받기
    - [ ] 로또 구입 금액 입력 받기
    - [ ] 당첨 번호 입력 받기
    - [ ] 보너스 번호 입력 받기
- [ ] 로또 번호 발행
- [ ] 당첨 결과 확인 및 저장
- [ ] 결과 출력
    - [ ] 발행한 로또 수량 및 번호 출력 (오름차순)
    - [ ] 당첨 내역 출력
- [ ] 입력값 검증
    - [ ] 구입 금액/당첨 번호/보너스 번호는 양수로 변환 가능해야 함
    - [ ] 구입 금액은 1000원으로 나누어 떨어져야 함
    - [ ] 당첨 번호는 쉼표를 기준으로 구분
    - [ ] 당첨 번호는 6개의 숫자여야 함
    - [ ] 당첨 번호는 1에서 45 사이의 숫자여야 함
